## FakeNet-NG — Start Before Detonating

```cmd
cd C:\Users\tasox\Downloads\fakenet3.5\fakenet3.5
fakenet.exe
```

FakeNet intercepts:
- **DNS** — returns `127.0.0.1` for all queries
- **HTTP/HTTPS** — serves a fake response
- **FTP** — simulates login
- **SMTP/IRC** — absorbs connections silently

All traffic is logged to the FakeNet console window and a PCAP file. Review the log after execution to extract C2 domains, IPs, and beaconing patterns.
