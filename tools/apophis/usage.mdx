## Folder Layout

| Folder | Description |
|--------|-------------|
| `Templates/` | Shellcode runners with and without encryption in various formats |
| `payloads/` | Generated shellcode runners ready to execute |
| `ConfuserEx/` | Compiled ConfuserEx files |
| `Net-Obfuscate/` | Compiled Net-Obfuscate files |

## Running Apophis

```bash
./apophis.sh
```

A successful run produces output like:

