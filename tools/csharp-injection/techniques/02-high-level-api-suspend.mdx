Demonstrates shellcode injection into `wuauclt.exe` (or any short-lived process). Suspends threads to keep the process open for analysis with tools like x64dbg.

## API Calls

- `OpenProcess`
- `VirtualAllocEx`
- `WriteProcessMemory`
- `CreateRemoteThread`
