---
title: "03 — Low Level API (Map Section)"
description: "Shellcode injection via NtCreateSection and NtMapViewOfSection."
---

Injection into a process of your choice (e.g. `explorer.exe`) using lower-level Windows APIs to create and map a new section into the remote process.

## API Calls

- `OpenProcess`
- `NtCreateSection`
- `NtMapViewOfSection`
- `NtCreateThreadEx` (or `CreateRemoteThread`)
