Classic shellcode injection (x86/x64) into a process of your choice (default: `notepad.exe`). Similar to technique 01 but uses additional process enumeration APIs.

## API Calls

- `CreateToolhelp32Snapshot`
- `Process32First`
- `Process32Next`
- `OpenProcess`
- `VirtualAllocEx`
- `WriteProcessMemory`
- `CreateRemoteThread`
- `WaitForSingleObject`
