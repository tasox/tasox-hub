A collection of process injection techniques using low and higher level Windows API calls. Supports both **x86/x64** architectures and outputs memory addresses, payload address, remote PID, and ThreadId for debugging.

<Info>
  [GitHub Repository](https://github.com/tasox/CSharp_Process_Injection)
</Info>

## Techniques

| # | Technique | API Level |
|---|-----------|-----------|
| 01 | High Level Windows API | High |
| 02 | High Level Windows API — Suspend spawned process | High |
| 03 | Low Level — Map a view of a section | Low |
| 04 | Low Level — Modify permissions of a section | Low |
| 05 | Low Level — NtAPIs only (1) | Low |
| 06 | Classic Injection using Shellcode | Classic |
| 07 | Classic DLL Injection | Classic |

## Prerequisites

- Visual Studio with **Allow Unsafe Code** enabled (Project Properties → Build)
- Match solution platform (x86/x64) to the target process architecture

## Windows API Reference

### Process Access Flags

```
All = 0x001F0FFF        CreateThread = 0x00000002
VirtualMemoryRead = 0x00000010    VirtualMemoryWrite = 0x00000020
QueryInformation = 0x00000400     Synchronize = 0x00100000
```

### Memory Protection

```
Execute = 0x10          ExecuteRead = 0x20
ExecuteReadWrite = 0x40 NoAccess = 0x01
ReadWrite = 0x04
```

### Allocation Type

```
Commit = 0x1000    Reserve = 0x2000
Physical = 0x400000    LargePages = 0x20000000
```
