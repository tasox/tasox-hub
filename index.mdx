## Welcome

A collection of open-source tools for **offensive security research**, **digital forensics & incident response (DFIR)**, and **malware analysis** — built for red teamers, threat hunters, and SOC analysts.

<Warning>
  These tools are intended for **authorized security research, penetration testing, and red team operations only**. Always obtain proper written authorization before using any of these tools in a target environment. Unauthorized use may violate local, national, or international laws.
</Warning>

## DFIR & Malware Analysis

Tools for incident response, forensic artifact extraction, and malware triage.

<CardGroup cols={3}>
  <Card
    title="dmpHunter"
    icon="search"
    color="#E63946"
    href="/tools/dmphunter/overview"
  >
    Hunt indicators inside Windows **mini/full** process dumps — memory maps, handles, threads, modules, YARA, strings, and network heuristics.

    **Language:** Python · **Platform:** Windows / Linux / macOS
  </Card>
  <Card
    title="accdu_inspector"
    icon="shield-alert"
    color="#E63946"
    href="/tools/accdu-inspector/overview"
  >
    Forensic analyzer for Microsoft Access databases — uncovers hidden VBA macros, OLE streams, and binary blobs with YARA scanning and suspicion scoring.

    **Language:** Python · **Platform:** Cross-platform
  </Card>
  <Card
    title="rarADSExtractor"
    icon="file-search"
    color="#E63946"
    href="/tools/raradsextractor/overview"
  >
    Safe Python-only RAR5 parser that carves **Alternate Data Streams** without invoking vulnerable binaries — built for CVE-2025-8088 / CVE-2025-6218 triage.

    **Language:** Python · **Platform:** Cross-platform
  </Card>
</CardGroup>

## AI-Powered Security

Tools that leverage AI for security research and intelligence workflows.

<CardGroup cols={1}>
  <Card
    title="triage-llm-lab"
    icon="flask"
    color="#457B9D"
    href="/tools/triage-llm-lab/overview"
  >
    Three-VM isolated sandbox combining Claude Code with REMnux, Ghidra, and VirusTotal — orchestrated via custom MCP servers for fully AI-driven malware triage and DFIR workflows.

    **Language:** TypeScript / Python · **Platform:** VMware (macOS / Linux)
  </Card>
  <Card
    title="TI Agent"
    icon="shield"
    color="#457B9D"
    href="/tools/ti-agent/overview"
  >
    Full-stack local app that fetches **97 built-in security RSS/Atom feeds**, passes real article content to Claude, and generates structured threat intelligence briefings with inline citations, IOC highlights, CVE tables, MITRE ATT&CK coverage, and 4 export formats.

    **Language:** Node.js / React · **Platform:** macOS / Linux
  </Card>
</CardGroup>

## Offensive Security

Tools for shellcode generation, payload delivery, and process injection research.

<CardGroup cols={2}>
  <Card
    title="Apophis"
    icon="terminal"
    color="#1D3557"
    href="/tools/apophis/overview"
  >
    Bash script leveraging DotNetToJScript, ConfuserEx, and Mono to generate shellcode runners in **10+ formats** — XOR, Caesar, 3DES, AES, MSI, HTA, JS, XSL, ASPX, and InstallUtil.

    **Language:** Bash / C# · **Platform:** Linux (Kali)
  </Card>
  <Card
    title="Browser Extension for Code Exec"
    icon="puzzle"
    color="#1D3557"
    href="/tools/browser-extension/overview"
  >
    Chrome/Edge extension that uses the **Native Messaging API** to bridge the browser and the host OS — demonstrated with Notepad but built for broader command execution scope.

    **Language:** Python / JS · **Platform:** Windows / macOS / Linux
  </Card>
</CardGroup>
